# Clean-Node
Proyecto de Autenticación(Login y Register) con Clean Architecture con NodeJS y Typescript

### Instalación

1. Clonar archivo **.env.template** y renombrar a **.env**.
2. Instalar dependencias.
```bash
npm install
```
3. Levantar la base de datos
```bash
docker-compose up -d
```

4. Ejecutar proyecto en modo desarrollo
```bash
npm run dev
```
